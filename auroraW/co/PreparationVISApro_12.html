Le LMS

Définition : LMS

Learning Management System, ou système de gestion des apprentissages.

Un LMS est un type de site web orienté formation. Il permet d'accéder aux cours, d'inscrire des formateurs et des apprenants, de gérer les sessions, de suivre les apprentissages, de vérifier le temps passé sur chaque activité...

Dans votre cas, le LMS vous permet d'accéder aux cours et exercices, et d'évaluer votre progression dans la formation.

Fondamental : Accéder au LMS

  • Aller sur Libres Savoirs

  • cliquer dans le menu du haut sur Se connecter

  • Donner son identifiant et son mot de passe

  • Cliquer dans le menu du haut sur Espace apprenant

  • Cliquer sur votre formation.

Une fois connecté sur le LMS Syfadys, cliquer sur l'intitulé de votre module puis "Démarrer la formation", une nouvelle fenêtre de votre navigateur s'ouvrira contenant le parcours de votre module de formation (ex : ci-dessous)

Ce document, accessible depuis la page d'accueil du LMS, peut vous guider dans votre utilisation.
